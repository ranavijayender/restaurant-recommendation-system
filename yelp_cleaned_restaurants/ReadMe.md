# Yelp Restaurant Data Mining Project

## Overview

This project applies data mining techniques to the Yelp Restaurant Dataset.

The project includes:

1. Data Understanding
2. Data Preprocessing
3. Exploratory Data Analysis
4. Restaurant Segmentation using K-Means Clustering
5. Restaurant Recommendation System using TF-IDF and Cosine Similarity

## Dataset

Yelp Restaurant Dataset

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Project Structure

1_understanding.ipynb

2_preprocessing.ipynb

3_analysis.ipynb

4_segmentation.ipynb

5_recommendation.ipynb

## Key Findings

- Most restaurants are rated between 3.5 and 4.5 stars.
- Philadelphia contains the highest number of restaurants.
- Three restaurant clusters were identified using K-Means.
- A recommendation system was developed using TF-IDF and Cosine Similarity.
